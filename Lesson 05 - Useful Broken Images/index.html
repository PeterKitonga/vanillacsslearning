In this lesson, I learnt about styling the browser default display of a broken or missing image.

Example #1
  1. .image {
  2.     position: relative;
  3.     display: inline-block;
  4.     font-size: 1rem;
  5.     text-align: center;

  8. }
  9.  
  10. .image::after {
  11.     z-index: 1;
  12.     display: inherit;
  13.     position: absolute;
  14.     top: 0;
  15.     left: 0;
  16.     width: 100%;
  17.     min-width: 300px;
  18.     padding: 10px 20px 10px 45px;
  19.     border: dashed 1px #888888;
  20.     background-color: #f0f0f0;
  21.     background-image: url(../../../assets/images/noimage.png);
  22.     background-size: 10%;
  23.     background-repeat: no-repeat;

  26.     *  More info: https://css-tricks.com/css-attr-function-got-nothin-custom-properties/
  27.     */
  28.     content: "" attr(alt) " Not Found";
