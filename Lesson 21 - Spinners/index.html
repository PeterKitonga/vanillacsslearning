In this lesson, I created a small triangle that, once hovered over, animates into a pyramid comprised of several shapes! Here I used the clip-path property.

Example #1
  1. .spinner-1 {
  2.     position: relative;
  3.     display: inline-block;
  4.     vertical-align: middle;
  5.     width: 64px;
  6.     height: 64px;
  7.     border-radius: 64px;
  8. }
  9.  
  10. .spinner-1::before {
  11.     position: absolute;
  12.     content: "";
  13.     width: 64px;
  14.     height: 64px;
  15.     border-radius: 64px;
  16.     color: #323b40;
  17.     box-shadow: inset -5px 0 0 5px currentColor;
  18.     clip: rect(0, 64px, 32px, 0);
  19.     animation: rotator 1s infinite ease;
  20. }
  21.  
  22. @keyframes rotator {
  23.     0% {
  24.         transform: rotate(-180deg);
  25.         box-shadow: inset -5px 0 0 5px currentColor;
  26.     }
  27.  
  28.     50% {
  29.         transform: rotate(0deg);
  30.         box-shadow: inset -1px 0 0 1px currentColor;
  31.     }
  32.  
  33.     100% {
  34.         transform: rotate(180deg);
  35.         box-shadow: inset -5px 0 0 5px currentColor;
  36.     }
  37. }
