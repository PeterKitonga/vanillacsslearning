In this lesson, I learnt about styling text. This involved adding fonts, sizing, rotating and adding shadows to text.

Cool Shadow

  1. .shadow-typo {
  2.     position: relative;
  3.     display: inline-block;
  4.     font-size: 6rem;
  5.     text-transform: uppercase;
  6.     color: #00b3b4;
  7.     /* The text-shadow property is described by X and Y offsets relative to the element, blur radius, and color. 
  8.     *  More info: https://css-tricks.com/almanac/properties/t/text-shadow/. 
  9.     *  By using a negative value for the Y offset it will apply a shadow above the text.
  10.     */
  11.     text-shadow: 3px 3px 0 #e7eef1, 8px 8px 0 rgba(0, 0, 0, .1);
  12. }

3D Type

  1. .threedee-typo {
  2.     font-size: 6rem;
  3.     /* The text-transform property controls text case and capitalization.
  4.     *  More info: https://css-tricks.com/almanac/properties/t/text-transform/
  5.     */
  6.     text-transform: uppercase;
  7.     color: #f2395a;
  8.     -webkit-transform: skew(-5deg, -5deg) rotate(5deg);
  9.     /* The transform property allows you to visually manipulate an element by skewing, rotating, translating, or scaling.
  10.     * More info: https://css-tricks.com/almanac/properties/t/transform/
  11.     */
  12.     transform: skew(-5deg, -5deg) rotate(5deg);
  13.     -webkit-transform-origin: center center;
  14.     /* The transform-origin property lets you change the point of origin of a transform and is used in conjunction with
  15.     *  the transform property when rotating, skewing, scaling e.t.c. The property can take up to two space-separated keyword 
  16.     *  or length values for a 2D transform and up to three values for a 3D transform.
  17.     *  More info: https://css-tricks.com/almanac/properties/t/transform-origin/
  18.     */
  19.     transform-origin: center center;
  20.     text-shadow: 1px 1px #d10e31,
  21.                  2px 2px #d10e31,
  22.                  3px 3px #d10e31,
  23.                  4px 4px #d10e31,
  24.                  5px 5px #d10e31,
  25.                  6px 6px #d10e31,
  26.                  7px 7px #d10e31,
  27.                  8px 8px #890920,
  28.                  9px 9px #890920,
  29.                  10px 10px #890920,
  30.                  11px 11px 15px rgba(0, 0, 0, .5);
  31. }
