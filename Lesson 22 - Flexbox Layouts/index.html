In this lesson, I briefly went through some concepts of CSS flexbox. CSS flexbox allows for a more efficient way of designing layouts, aligning and distributing space among items in a container. For more info visit here.

Flexbox Layout #1
  1. .layout1 main {
  2.     /*
  3.     * The 'flex' value is used to tell the browser that this element is a flex container.
  4.     */
  5.     display: flex;
  6.     /*
  7.     * This property defines the direction flex items are placed in the flex container. This
  8.     * property has 4 values namely: row, row-reverse, column and column-reverse. 
  9.     * More info: https://css-tricks.com/almanac/properties/f/flex-direction/
  10.     */
  11.     flex-direction: row;
  12. }
  13.  
  14. .layout1 article {
  15.     /*
  16.     * The flex property is a shorthand property with parameters of flex-grow, flex-shrink and flex-basis respectively. The
  17.     * last two parameters are optional.
  18.     * More info: https://css-tricks.com/almanac/properties/f/flex/
  19.     */
  20.     flex: 1 58%;
  21.     margin-right: 2% !important;
  22. }
