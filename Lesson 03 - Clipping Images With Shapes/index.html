Clipping Images With Shapes

Next >>

In this lesson, I learnt about styling images. This involved creating shapes with images using clip paths.

Example #1
  1. .clip-image-1 {
  2.     -webkit-clip-path: circle(50% at 50% 50%);
  3.     /* The clip-path property allows you to specify a specific region of an element to display, rather than showing the complete area.
  4.     *  More info: https://css-tricks.com/almanac/properties/c/clip-path/
  5.     *  It is commonly used with images but is not limited to them only.
  6.     *
  7.     *  The points for each shape defined by this property are defined by X and Y offsets respectively. For example below illustration
  8.     *  creates a circle centered at 50% of the image with an X and Y offset of 50% each.
  9.     */
  10.     clip-path: circle(50% at 50% 50%);
  11. }
