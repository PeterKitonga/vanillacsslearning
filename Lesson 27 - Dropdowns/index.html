In this lesson, I learnt how to design a dropdown using CSS only and no JavaScript. The dropdown triggers when one hovers on the dropdown button.

Dropdown Example
  1. .dropdown {
  2.     position: relative;
  3.     display: inline-block;
  4.     z-index: 10;
  5. }
  6.  
  7. .dropdown__hover {
  8.     display: inline-block;
  9.     padding: 10px;
  10.     background: #323b40;
  11.     color: white;
  12.     cursor: pointer;
  13.     border: none;
  14. }
  15.  
  16. .dropdown__hover:hover {
  17.     background: #222222;
  18. }
  19.  
  20. .dropdown__menu {
  21.     position: absolute;
  22.     display: none;
  23.     top: 100%;
  24.     left: 0;
  25.     min-width: 200px;
  26.     background: white;
  27.     color: #323b40;
  28.     box-shadow: 0 3px 2px rgba(0, 0, 0, .3);
  29. }
  30.  
  31. .dropdown:hover .dropdown__menu {
  32.     display: block;
  33. }
  34.  
  35. .dropdown__menu a {
  36.     display: block;
  37.     border: none;
  38.     padding: 10px;
  39.     text-decoration: none;
  40.     border-top: solid 1px #e0e0e0;
  41. }
  42.  
  43. .dropdown__menu a:hover {
  44.     background: #f0f0f0;
  45.     color: #323b40;
  46. }
