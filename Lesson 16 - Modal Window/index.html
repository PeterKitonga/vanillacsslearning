In this lesson, I learnt creating modal window with CSS. Here, I used a checkbox to track the state of opening and closing the modal.

Modal Window
  1. .overlay {
  2.     position: fixed;
  3.     top: 0;
  4.     right: 0;
  5.     bottom: 0;
  6.     left: 0;
  7.     z-index: 99;
  8.     background: #000000;
  9.     /*
  10.     * The pointer-events property allows for control over how HTML elements respond to mouse/touch events.
  11.     * More info: https://css-tricks.com/almanac/properties/p/pointer-events/
  12.     */
  13.     pointer-events: none;
  14.     opacity: 0;
  15.     -webkit-transform: scale(0.5);
  16.     transform: scale(0.5);
  17. }
  18.  
  19. input[type="checkbox"]:checked ~ .overlay {
  20.     pointer-events: auto;
  21.     opacity: .8;
  22.     -webkit-transform: scale(1);
  23.     transform: scale(1);
  24.     -webkit-transition: transform .5s ease, opacity .5s ease;
  25.     transition: transform .5s ease, opacity .5s ease;
  26. }
  27.  
  28. .modal-css {
  29.     position: fixed;
  30.     top: 50%;
  31.     left: 50%;
  32.     z-index: 100;
  33.     -webkit-transform: translate(-50%, -50%) scale(.5);
  34.     transform: translate(-50%, -50%) scale(.5);
  35.     width: 100%;
  36.     max-width: 640px;
  37.     pointer-events: none;
  38.     opacity: 0;
  39. }
  40.  
  41. input[type="checkbox"]:checked ~ .modal-css {
  42.     pointer-events: auto;
  43.     opacity: 1;
  44.     -webkit-transform: translate(-50%, -50%) scale(1);
  45.     transform: translate(-50%, -50%) scale(1);
  46.     -webkit-transition: transform .5s ease, opacity .5s ease;
  47.     transition: transform .5s ease, opacity .5s ease;
  48.     transition-delay: .1s;
  49. }
  50.  
  51. .modal__content {
  52.     display: block;
  53.     padding: 30px;
  54.     background: #ffffff;
  55. }
