In this lesson, I learnt about designing sliders that display more content over an image. This can be used showcase one's portfolio on their website and provide links when hovering over the image.

General Markup
  1. .slide {
  2.     position: relative;
  3.     display: inline-block;
  4.     overflow: hidden;
  5. }
  6.  
  7. .slide img {
  8.     display: inline-block;
  9.     vertical-align: middle;
  10. }
  11.  
  12. .slide-content {
  13.     position: absolute;
  14.     top: 0;
  15.     left: 0;
  16.     right: 0;
  17.     bottom: 0;
  18.     width: 100%;
  19.     height: 100%;
  20.     z-index: 1;
  21.     display: flex;
  22.     /*
  23.     * The align-items property affects how elements are aligned
  24.     * in flexbox and grid layouts. In flexbox there are 5 possible
  25.     * values: flex-start | flex-end | center | baseline | stretch
  26.     * More info: https://css-tricks.com/almanac/properties/a/align-items/
  27.     */
  28.     align-items: center;
  29.     /*
  30.     * The justify-content property helps distribute extra free space leftover
  31.     * when either all the flex items on a line are inflexible, or are flexible
  32.     * but have reached their maximum size.
  33.     * More info: https://css-tricks.com/almanac/properties/j/justify-content/
  34.     */
  35.     justify-content: center;
  36.     text-align: center;
  37.     background-color: rgba(225, 35, 69, .9);
  38.     padding: 20px;
  39.     color: white;
  40.     box-sizing: border-box;
  41.     transition: transform .3s ease;
  42. }
  43.  
  44. .slide:hover .slide-content {
  45.     transform: translate(0, 0);
  46. }
