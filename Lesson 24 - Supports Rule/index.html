In this lesson, I learnt about the @supports feature in CSS which checks if a property is supported by your browser before applying a set of styles to an element.

@supports Rule Example #1
  1. .circle {
  2.     height: 100px;
  3.     width: 100px;
  4.     line-height: 100px;
  5.     background-color: #28a745;
  6.     color: white;
  7.     text-align: center;
  8. }
  9.  
  10. /*
  11. * The @supports feature tests if the browser supports a particular property or 
  12. * property: value combination before applying a block of styles.
  13. * More info: https://css-tricks.com/how-supports-works/
  14. */
  15. @supports (border-radius: 100px) {
  16.     .circle {
  17.         border-radius: 100px;
  18.     }
  19. }

Do you see a circle? Your browser is good.

Don't see a circle? The @supports property knows that, and didn't serve up that style for you.
