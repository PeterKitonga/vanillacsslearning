In this lesson, I learnt how style layouts by adding floats and fixing elements that collapse when floats are applied.

Modern Layout #1
  1. .layout1 article {
  2.     width: 58%;
  3.     float: left;
  4. }
  5.  
  6. .layout1 aside {
  7.     width: 40%;
  8.     float: right;
  9. }
  10.  
  11. .layout1 footer {
  12.     /* The clear property is used in CSS to move an element below elements that are floated on either of its sides.
  13.     *  More info: https://css-tricks.com/almanac/properties/c/clear/
  14.     */
  15.     clear: both;
  16. }
