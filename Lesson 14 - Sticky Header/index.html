In this lesson, I learnt about styling a header to make it remain at the top of the page.

Sticky Header
  1. .sticky-header-lesson {
  2.     margin-top: 200px;
  3. }
  4.  
  5. .sticky-header {
  6.     background: rgba(242, 57, 90, .9);
  7.     position: fixed;
  8.     top: 0;
  9.     height: 180px;
  10.     width: 100%;
  11.     z-index: 99;
  12.     box-shadow: 0 0 10px rgba(0, 0, 0, .2);
  13.     transition: all .1s ease-in-out;
  14. }
  15.  
  16. .sticky-header h1 {
  17.     font-size: 2.8rem;
  18.     line-height: 3.2rem;
  19.     margin: 0 auto;
  20. }
  21.  
  22. .sticky-header h1 small {
  23.     font-size: 2rem;
  24. }
At the top is the sticky header with a defined height and 100% width. The header remain stuck at the top as you scroll and collapses into a smaller height.
Sticky Header (After scroll)
  1. .sticky-header-scrolled {
  2.     height: 90px !important;
  3. }
  4.  
  5. .sticky-header-scrolled h1 {
  6.     font-size: 1.8rem;
  7.     line-height: 1.2rem;
  8.     margin: 0 auto;
  9. }
  10.  
  11. .sticky-header-scrolled h1 small {
  12.     font-size: 1rem;
  13. }
With this CSS we define the new height of the header after the scroll event hits 200px. The font size and line height of the header text is resized as well.
jQuery
  1. $(function () {
  2.      // Listens for the scroll event of the window
  3.     $(window).scroll(function() {
  4.        // Gets the scroll position of the window
  5.        var height = $(window).scrollTop();
  6.  
  7.        // Adds/Removes the 'sticky-header-scrolled' class from the header element depending on the scroll position
  8.        if(height  >= 200) {
  9.            $('.sticky-header').addClass('sticky-header-scrolled');
  10.        } else {
  11.            $('.sticky-header').removeClass('sticky-header-scrolled');
  12.        }
  13.     });
  14. });
This logic assits to keep track of the window's scroll position and add/removes a custom class for the header.
