In this lesson, I learnt to make and design progress bars. This lesson further expounded on using CSS animations.

Example #1
  1. .animation-bar-1 {
  2.     position: relative;
  3.     display: block;
  4.     line-height: 16px;
  5.     padding: 5px;
  6.     font-size: 16px;
  7.     border-radius: 30px;
  8.     background: rgba(0, 0, 0, .1);
  9.     box-shadow: inset 0 1px 2px rgba(0, 0, 0, .25), 0 1px 0 rgba(255, 255, 255, .8);
  10. }
  11.  
  12. .animation-bar-1 span {
  13.     position: relative;
  14.     display: inline-block;
  15.     vertical-align: middle;
  16.     height: 20px;
  17.     border-radius: 10px 0 0 10px;
  18.     overflow: hidden;
  19.     background-color: #F56982;
  20.     background-size: 100%;
  21.     background-image: linear-gradient(to bottom, #F2395A, #B90C2B);
  22.     animation: progress-bar-1 6s infinite ease;
  23. }
  24.  
  25. .animation-bar-1 span::after {
  26.     position: absolute;
  27.     top: 0;
  28.     right: 0;
  29.     bottom: 0;
  30.     left: 0;
  31.     content: "";
  32.     background-size: 100%;
  33.     background-image: linear-gradient(
  34.         45deg, 
  35.         #FFFFFF 25%,
  36.         rgba(0, 0, 0, 0) 25%,
  37.         rgba(0, 0, 0, 0) 50%,
  38.         #FFFFFF 50%,
  39.         #FFFFFF 75%,
  40.         rgba(0, 0, 0, 0) 75%,
  41.         rgba(0, 0, 0, 0)
  42.     );
  43.     background-size: 30px 30px;
  44.     opacity: .3;
  45.     animation: progress-bar-1-after .5s infinite linear;
  46. }
  47.  
  48. @keyframes progress-bar-1 {
  49.     0% {
  50.         width: 0%;
  51.     }
  52.  
  53.     100% {
  54.         width: 60%;
  55.     }
  56. }
  57.  
  58. @keyframes progress-bar-1-after {
  59.     0% {
  60.         /*
  61.         * This property allows you to move a background image or gradient around within its container. Default values(x y) are 0 0
  62.         * which is the top left of the container.
  63.         * More info: https://css-tricks.com/almanac/properties/b/background-position/
  64.         */
  65.         background-position: 0 100%;
  66.     }
  67.  
  68.     100% {
  69.         background-position: 30px 100%;
  70.     }
  71. }
