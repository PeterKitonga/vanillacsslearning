In this lesson, I learnt about styling images to create various visual effects using filters.

Example #1
  1. .grayscale {
  2.     /* The filter property is used to achieve various visual effects similar to the ones on photoshop (filters) to images.
  3.     *  These effects include: grayscale, invert, blur, drop-shadow, sepia, contrast, saturate
  4.     *  More info: https://css-tricks.com/almanac/properties/f/filter/
  5.     */
  6.     -webkit-filter: grayscale(1);
  7.     filter: grayscale(1);
  8. }
  9.  
  10. .grayscale:hover {
  11.     -webkit-filter: grayscale(0);
  12.     filter: grayscale(0);
  13. }
