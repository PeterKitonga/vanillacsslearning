Depth of Field Blur

Next >>

In this lesson, I learnt about creating a depth of field on a gallery of images. Here we would blur the images when one hovers over the container containing the images and play around transitioning with scaling the images.

Example #1
  1. .depth-gallery {
  2.     position: relative;
  3.     display: inline-block;
  4. }
  5.  
  6. .depth-gallery img {
  7.     margin: 2px;
  8.     width: 345px;
  9.     cursor: pointer;
  10.     box-sizing: border-box;
  11.     border: solid 7px white;
  12.     box-shadow: 0 0 5px rgba(0, 0, 0, .3);
  13.     transition: filter .3s linear, transform .3s linear;
  14. }
  15.  
  16. .depth-gallery:hover img {
  17.     filter: blur(2px);
  18.     transform: scale(.9);
  19. }
  20.  
  21. .depth-gallery img:hover {
  22.     filter: blur(0);
  23.     transform: scale(1);
  24. }
