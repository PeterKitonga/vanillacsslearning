In this lesson, I learnt about conditional comments that are used to style websites for use with Internet Explorer.

Variables
  1. .bg-light {
  2.     --primary-color: orangered;
  3.     --secondary-color: hotpink;
  4.     --basic-measurement: 20px;
  5. }
The above vairables are scoped within the bg-light wrapper and will not be usable outside the wrapper.
Example #1
  1. .bg-light .box {
  2.     text-align: center;
  3.     --text-color: #f3f3f3;
  4.     /* The var() function in native CSS is used to select variables created with the (--) prefix. */
  5.     color: var(--text-color);
  6.     background: var(--primary-color);
  7.     margin: var(--basic-measurement) auto;
  8.     /* The calc() property value is a CSS function used for doing basic math.
  9.     *  More info: https://css-tricks.com/a-complete-guide-to-calc-in-css/
  10.     */
  11.     height: calc(var(--basic-measurement) * 10);
  12.     width: calc(var(--basic-measurement) * 10);
  13.     line-height: calc(var(--basic-measurement) * 10);
  14. }
