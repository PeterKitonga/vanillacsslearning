In this lesson, I learnt to make and design tooltips.

Tooltip General Markup
  1. .tooltip-custom {
  2.     position: relative;
  3.     height: 60px;
  4.     width: 250px;
  5.     background: #F2395A;
  6.     color: white;
  7.     text-align: center;
  8.     /*
  9.     * Notice how applying a line-height similar to the element's height 
  10.     * vertically aligns the inline elements within it.
  11.     */ 
  12.     line-height: 60px;
  13.     font-size: 18px;
  14.     margin: 0 auto;
  15. }
  16.  
  17. .tooltip-custom::before {
  18.     position: absolute;
  19.     z-index: 2;
  20.     display: none;
  21.     width: 200px;
  22.     padding: 10px;
  23.     font-size: .8em;
  24.     line-height: 1.4em;
  25.     border-radius: 5px;
  26.     background-color: rgba(0, 0, 0, .8);
  27.     color: white;
  28.     content: attr(data-title);
  29. }
  30.  
  31. .tooltip-custom::after {
  32.     position: absolute;
  33.     z-index: 1;
  34.     display: none;
  35.     width: 0;
  36.     height: 0;
  37.     border: 5px solid transparent;
  38.     content: "";
  39. }
  40.  
  41. .tooltip-custom:hover::before, 
  42. .tooltip-custom:hover::after {
  43.     display: inline-block;
  44. }
Here, I styled the actual element that one would hover on to trigger the tooltip.
