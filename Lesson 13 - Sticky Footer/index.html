Sticky Footer

Next >>

In this lesson, I learnt about styling a footer to make it remain at the bottom of the page even when there is no content in the viewport.

Example #1
  1. html, body {
  2.     height: 100%;
  3.     margin: 0;
  4. }
  5.  
  6. .page-content {
  7.     min-height: 100%;
  8.  
  9.     /* negative value of the margin is same measurement as footer height */
  10.     margin-bottom: -120px;
  11. }
  12.  
  13. .page-content::after {
  14.     content: "";
  15.     display: block;
  16. }
  17.  
  18. footer, .page-content::after {
  19.     height: 120px;
  20. }
At the bottom is the sticky footer that depending on the height of the page content wapper will remain stuck to the bottom of the page.
This is a sticky footer! Resize the height of the browser, watch what happens!
