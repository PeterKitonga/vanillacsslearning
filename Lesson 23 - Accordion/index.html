In this lesson, I learnt about designing an accordion using CSS3. Accordions can be used to hide/show content depending on the title a user has clicked.

Example #1
  1. .accordion {
  2.     position: relative;
  3.     display: block;
  4.     background-color: white;
  5.     border: solid 1px #e0e0e0;
  6. }
  7.  
  8. .accordion__item {
  9.     display: block;
  10. }
  11.  
  12. .accordion .accordion__content {
  13.     display: block;
  14.     padding: 0;
  15.     height: 0;
  16.     overflow: hidden;
  17.     transition: height .5s ease, padding .3s linear;
  18. }
  19.  
  20. .accordion .accordion__content p {
  21.     padding: 20px;
  22.     margin: 0;
  23. }
  24.  
  25. /*
  26. * Here we can use the :target pseudo selector to toggle the accordion like so:
  27. *
  28. * .accordion .accordion__content:target {
  29. *     height: 150px;
  30. * }
  31. *
  32. * The :target pseudo selector in CSS matches when the hash in the URL and the id of 
  33. * an element are the same.
  34. * More info: https://css-tricks.com/almanac/selectors/t/target/
  35. */
  36. .accordion .accordion__content.accordion__toggled {
  37.     height: 150px;
  38. }
  39.  
  40. .accordion .accordion__trigger {
  41.     text-decoration: none;
  42. }
  43.  
  44. .accordion .accordion__title {
  45.     position: relative;
  46.     display: block;
  47.     margin: 0;
  48.     padding: 10px;
  49.     font-size: 1em;
  50.     background: #00b3b4;
  51.     color: white;
  52.     border: solid 1px #379aa4;
  53.     box-shadow: 0 1px 0 rgba(255, 255, 255, .5) inset, 0 -1px 0 rgba(255, 255, 255, .1) inset, 0 4px 0 #338A94, 0 4px 2px rgba(0, 0, 0, .5);
  54. }
  55.  
  56. .accordion .accordion__title:hover {
  57.     background-color: #0bbfc0;
  58. }
  59.  
  60. .accordion .accordion__title::after {
  61.     position: absolute;
  62.     content: "";

  66.     height: 7px;
  67.     border-left: solid 2px white;
  68.     border-bottom: solid 2px white;

  70. }

JavaScript
  1. <script>
  2.         var app = {};
  3.  
  4.         // select all accordion trigger elements
  5.         app.accordions = document.querySelectorAll('.accordion .accordion__trigger');
  6.  
  7.         // listen for click events on all the accordion trigger elements
  8.         app.accordions.forEach(function (accordion) {
  9.             accordion.addEventListener('click', function(event) {
  10.                 // prevents default action of scrolling to the position of the clicked accordion
  11.                 event.preventDefault();
  12.  
  13.                 // gets the data-toggle attribute of the clicked accordion
  14.                 var toggle = event.target.getAttribute('data-toggle');
  15.  
  16.                 // gets the target of the clicked accordion
  17.                 var toggle_element = document.getElementById(toggle);
  18.  
  19.                 // toggle the class of the clicked accordion and show content
  20.                 toggle_element.classList.toggle('accordion__toggled');
  21.             });
  22.         });
  23. </script>
Here, I added JavaScript to listen for the on click event of the accordion inorder to toggle the accordion content.
