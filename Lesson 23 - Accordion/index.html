In this lesson, I learnt about designing an accordion using CSS3. Accordions can be used to hide/show content depending on the title a user has clicked.

Example #1
  1. .accordion {
  2.     position: relative;
  3.     display: block;
  4.     background-color: white;
  5.     border: solid 1px #e0e0e0;
  6. }
  7.  
  8. .accordion__item {
  9.     display: block;
  10. }
  11.  
  12. .accordion .accordion__content {
  13.     display: block;
  14.     padding: 0;
  15.     height: 0;
  16.     overflow: hidden;
  17.     transition: height .5s ease, padding .3s linear;
  18. }
  19.  
  20. .accordion .accordion__content p {
  21.     padding: 20px;
  22.     margin: 0;
  23. }
  24.  
  25. /*
  26. * The :target pseudo selector in CSS matches when the hash in the 
  27. * URL and the id of an element are the same.
  28. * More info: https://css-tricks.com/almanac/selectors/t/target/
  29. */
  30. .accordion .accordion__content:target {
  31.     height: 150px;
  32. }
  33.  
  34. .accordion .accordion__trigger {
  35.     text-decoration: none;
  36. }
  37.  
  38. .accordion .accordion__title {
  39.     position: relative;
  40.     display: block;
  41.     margin: 0;
  42.     padding: 10px;
  43.     font-size: 1em;
  44.     background: #00b3b4;
  45.     color: white;
  46.     border: solid 1px #379aa4;
  47.     box-shadow: 0 1px 0 rgba(255, 255, 255, .5) inset, 0 -1px 0 rgba(255, 255, 255, .1) inset, 0 4px 0 #338A94, 0 4px 2px rgba(0, 0, 0, .5);
  48. }
  49.  
  50. .accordion .accordion__title:hover {
  51.     background-color: #0bbfc0;
  52. }
  53.  
  54. .accordion .accordion__title::after {
  55.     position: absolute;
  56.     content: "";
  57.     top: 50%;
  58.     right: 20px;
  59.     width: 7px;
  60.     height: 7px;
  61.     border-left: solid 2px white;
  62.     border-bottom: solid 2px white;
  63.     transform: rotate(-45deg) translateY(-50%);
  64. }
