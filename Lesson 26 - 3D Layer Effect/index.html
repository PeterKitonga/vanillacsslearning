In this lesson, I learnt about creating a 3d perspective of images. This perspective is created when hovering on the container with the images and expounds on using translations to achieve this.

3D Layers
  1. .layers {
  2.     position: relative;
  3.     min-height: 640px;
  4.     max-width: 360px;
  5. }
  6.  
  7. .layers img {
  8.     position: absolute;
  9.     max-width: 100%;
  10.     height: 100%;
  11.     transition: all 1600ms ease;
  12.     /*
  13.     * The transform-style property, when applied to an element, determines 
  14.     * if that elementâ€™s children are positioned in 3D space, or flattened.
  15.     * More info: https://css-tricks.com/almanac/properties/t/transform-style/
  16.     */
  17.     transform-style: preserve-3d;
  18. }
  19.  
  20. .layers img.mid {
  21.     width: 320px;
  22.     left: 20px;
  23. }
  24.  
  25. .layers img.top {
  26.     width: 300px;
  27.     left: 30px;
  28.     top: -5px;
  29. }
  30.  
  31. .layers img.tippy {
