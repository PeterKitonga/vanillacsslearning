Animated Pyramid

Next >>

In this lesson, I created a small triangle that, once hovered over, animates into a pyramid comprised of several shapes! Here I used the clip-path property.

Example #1
  1. .triangle {
  2.     position: relative;
  3.     height: 320px;
  4.     clip-path: polygon(
  5.         50% 0,
  6.         20% 100%,
  7.         80% 100%
  8.     );
  9. }
  10.  
  11. .triangle div {
  12.     position: absolute;
  13.     top: 0;
  14.     background: red;
  15.     width: 100%;
  16.     height: 100px;
  17.     margin: 0 auto;
  18.     color: red;
  19.     line-height: 100px;
  20.     text-align: center;
  21.     font-size: .8em;
  22.     transition: all ease .6s;
  23. }
  24.  
  25. .top {
  26.     z-index: 2;
  27. }
  28.  
  29. .mid {
  30.     z-index: 1;
  31. }
  32.  
  33. .triangle:hover div {
  34.     color: white;
  35. }
  36.  
  37. .triangle:hover .mid {
  38.     top: 110px;
  39.     background: green;
  40. }
  41.  
  42. .triangle:hover .bottom {
  43.     top: 220px;
  44.     background: blue;
  45. }
